Secciones repetidas con mustache

Las secciones de repetición son útiles cuando se desea mostrar una lista o tabla de datos relacionados en la página.

vamos a hacer aqui los siguiente

  • Agregamos una sección con mustache: {{#mySector}} ....... {{/mySector}}
  • Dentro de la sección, colocamos los elementos y variables que deseamos mostrar para cada lista
  • Agregamos una matríz de datos. Damos a mySector el mismo nombre de la sección
  • Cada elemento de la lista debe ser un objeto que contenga las propiedades correspondientes a las variables en el paso 2
